# DataToolBox
<br>This is a windows forms Data Tool Box solution for Infusionsoft. It is coded in C# and R. It contains: 
<br>Updater (for future updates)
<br>Remove Blank Columns from CSV and Change format to UTF-8
<br>CSV Splitter by any size (Splits 1 file into multiple by size, does not work with new lines)
<br>Salesforce Formatter (Removes all unnecessary columns and formats the data to work with Infusionsoft)
